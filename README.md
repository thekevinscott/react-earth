# React Earth

This is a package to display an Earth via a React component.

Image of it working.

You can see a live example here.

## Installation

Install with npm:
```
npm install react-earth
```

Or yarn:
```
yarn add react-earth
```

## Usage

```
import ReactEarth from 'react-earth';

const App = () => {
  return (
    <ReactEarth />
  );
};
```
